Must‑Have OBS Studio Plugins for Windows

Supercharge your workflow with tested, widely‑used plugins. Always download from trusted sources and make sure the build matches your OBS version and architecture (x64).

Popular plugins to consider

  • Move Transition: Smoothly animate sources and scenes with motion.
  • StreamFX: Advanced filters (blur, 3D transform, SDF effects). Check version compatibility for OBS 28+.
  • OBS Shaderfilter (Plus): Add custom shader effects to sources.
  • Audio Monitor: Route audio to monitors with more control.
  • Downstream Keyer / Source Dock: Flexible docks and keying workflows.

Tip: Browse the official OBS Forum Plugins for vetted releases.

Compatibility checklist

  • OBS 28+ uses Qt6; older plugin builds may not work.
  • Use x64 plugin builds for Windows 64‑bit.
  • Prefer plugins with active maintenance and clear release notes.
  • Back up your OBS profile/scene collection before testing new plugins.

How to install plugins (Windows)

  1. Download a plugin from a trusted source (OBS Forum/GitHub). If an installer (.exe) is provided, use it.
  2. Close OBS before installing. Run the installer with admin rights.
  3. Manual ZIP: extract and copy binaries to C:\Program Files\obs-studio\obs-plugins\64bit and data to C:\Program Files\obs-studio\data\obs-plugins\<plugin>.
  4. Launch OBS → check Tools, Filters, or the relevant menu for the plugin.

Quick FAQ

Q: Where is the plugins folder?
A: C:\Program Files\obs-studio\obs-plugins\64bit and ...\data\obs-plugins\<plugin> for data files.

Q: OBS crashes after adding a plugin?
A: Remove the plugin from the folder, relaunch OBS, and install a build compatible with your version.

Q: Do I need obs-websocket?
A: It's built‑in on OBS 28+; only install separately if specifically required.
